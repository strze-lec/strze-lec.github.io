Language School Database

Project prepared for the Intro to Data MOOC (DataQuest) Challenge organized by Code First Girls, which was recognized as one of three finalists' projects.

The database shows language school database with 17 tables in its original version. The school's database allows its viewers to check students' and teachers' finances, look at class statistics and see who has to put in extra work before they're allowed to participate in the certification process. The most interesting parts of the project are further explained in the presentation (linked in the GitHub describtion).
